# HANAX-Lab
And this is a high-end educational tool for physics, electricity simulator, where high school students build circuits, ask ai, get generated exercises with levels, game mode and visual laws using svg, and three languages.

NOTE : UI FOR COMPUTERS/LAPTOPS,NOT OPTIMIZED FOR MOBILE

Used by my school. Helps students visualize circuits before labs.

How: Wire components, see the output in real-time

Tech: Cloudflare Workers, SVG, HTML/CSS/JS

Status: Live at https://an-entire-kingdom-of-light-sounds-luminous.hanap1ec.workers.dev/
